Random Forest Classification

This project is a Machine Learning web application built using Python, Streamlit, and Scikit-learn. The application allows users to upload a CSV dataset, select a target column, and train a Random Forest Classifier model to predict outcomes.

Features
Upload CSV dataset
Automatic data preprocessing
Handle missing values and duplicates
Encode categorical data
Train Random Forest Classification model
Display model accuracy
Simple and interactive Streamlit interface

Technologies Used
Python
Streamlit
Pandas
Matplotlib
Scikit-learn

Files
app1.py → Streamlit web application
randomforest.ipynb → Jupyter Notebook implementation
requirements.txt → Required Python libraries

Run the Project

Install dependencies:

pip install -r requirements.txt

Run the Streamlit app:

streamlit run app1.py

Output
The application displays:
Original dataset
Dataset shape
Trained model accuracy
Classification result
